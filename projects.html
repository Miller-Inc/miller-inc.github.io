Miller Inc Physics Engine

This is an open source physics engine that utilizes multiprocessing compatible with a wide variety of devices. Prioritizing accuracy with , this engine is made to be compatible with just about every system (Windows/Linux) and able to utilize every bit of your machine's compute power. This project is written in C++ with a mix of CUDA and other GPU languages added as more compatibility is worked into the system. To find out more about this work-in-progress, see its repository at MillerInc/MillerInc.PhysicsEngine

The Remnant

This is the start to a Miller Inc original game developed with Unreal Engine 5, created with wholesome entertainment backed by a strong storyline. As it is in development, the code must remain private. To find out more about this project, check out its project page with details at MillerInc/remnant

Star Wars Card Game

This is a game that is a remake of a game that the founder of Miller Inc and his brothers had played when younger. Mixed with the mechanics of another game known as Star Wars Galaxy of Heroes, by Capital Games and EA, this game is going to offer many new experiences that were never offered in either of those games, including real-time combat with other players and a free-play mode. While in development, the source code is made available at MillerInc/Star-Wars-Card-Game
